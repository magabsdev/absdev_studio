# Frequently Asked Questions

## Is ABSDEV Studio a general-purpose IDE?

No. It is focused on professional Laravel and PHP development on macOS.

## Are PHP settings global?

No. Configure PHP per project from **Edit Project…**.

## Why can a tool be hidden?

Tools appear when the selected project supports them. Sail, containers and package-specific tools are capability-driven.

## Where is help stored?

The `Documentation/` folder is the source of truth and is bundled with the application.
