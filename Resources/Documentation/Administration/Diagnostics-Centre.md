# Diagnostics Centre

## Purpose

Unified diagnostics collect application, project, runtime, database, Git, AI, MCP, and documentation issues.

## How to use it

1. Open diagnostics from the relevant tool.
2. Filter by domain and severity.
3. Export evidence before clearing entries.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
