# Logging

## Purpose

Unified logging provides separate channels for application, project, runtime, database, Git, AI, MCP, and documentation activity.

## How to use it

1. Open Logs.
2. Filter to the relevant channel.
3. Export only after reviewing for secrets.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
