# Laravel Studio

## Purpose

Laravel Studio is the framework-aware project hub showing indexed routes, controllers, models, migrations, jobs, events, views, tests, packages, and configuration.

## How to use it

1. Select a Laravel project.
2. Open Laravel Studio.
3. Select a metric card to open the corresponding tool.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
