# Vite and Frontend

## Purpose

Frontend controls help run and inspect the Laravel Vite development workflow.

## How to use it

1. Confirm Node and package-manager paths in Runtime Centre.
2. Open Frontend.
3. Run the appropriate development or build command.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
