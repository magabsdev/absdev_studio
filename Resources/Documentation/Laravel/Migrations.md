# Migrations

## Purpose

Review and run Laravel migration operations against the selected project and configured environment.

## How to use it

1. Confirm the project and database first.
2. Open Migrations.
3. Review pending operations before executing destructive commands.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
