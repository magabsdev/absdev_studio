# PHP and Composer Problems

## Purpose

Resolve missing executables, incompatible PHP versions, Composer failures, and GUI PATH differences.

## How to use it

1. Open Runtime Centre.
2. Enter explicit executable paths.
3. Validate before retrying the command.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
