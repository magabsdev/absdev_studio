# AI Providers

## Purpose

ABSDEV Studio can work with configured AI providers, LM Studio, Open WebUI, and MCP-backed project intelligence.

## How to use it

1. Enable AI in Settings.
2. Configure the chosen provider.
3. Test the connection before using project-aware features.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
