# AI Documentation Writer

## Purpose

Generate project documentation from the Digital Twin and selected source context.

## How to use it

1. Choose the target area.
2. Generate into a reviewable document.
3. Verify generated statements against the code.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
