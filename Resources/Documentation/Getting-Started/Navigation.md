# Navigation

## Purpose

Use the project navigator to select a Laravel project and the tool navigator to open a feature.

## How to use it

1. Select a project before running project-aware tools.
2. Drag tool entries to reorder them.
3. Use the command palette for keyboard-first navigation.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
