# Observability

## Purpose

Observability integrates supported Laravel monitoring packages and application signals.

## How to use it

1. Install and configure a supported package.
2. Open Observability.
3. Correlate metrics with logs and runtime state.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
