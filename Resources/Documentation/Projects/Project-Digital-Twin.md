# Project Digital Twin

## Purpose

The Digital Twin is the shared Laravel project index used by Laravel Studio, AI, diagnostics, documentation, and analysis tools.

## How to use it

1. Open Laravel Studio to trigger indexing.
2. Use Refresh Index after large external changes.
3. Review indexed counts before relying on generated analysis.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
