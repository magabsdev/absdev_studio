# Project Editor

## Purpose

Edit a project alias, folder, icon, colour, and project-specific PHP runtime without renaming the source folder.

## How to use it

1. Select a project.
2. Open Edit Project from the context menu or sidebar control.
3. Save the alias and runtime settings.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
