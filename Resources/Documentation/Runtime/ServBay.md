# ServBay

## Purpose

ServBay integration provides local service, website, SSL, and runtime information when ServBay is installed.

## How to use it

1. Open ServBay from the navigator.
2. Confirm the selected project maps to the expected site.
3. Use project-specific PHP rather than assuming a global ServBay version.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
