# Node and Package Managers

## Purpose

Store Node, npm, pnpm, and Bun executable paths independently for each project.

## How to use it

1. Open Runtime Centre.
2. Use Detect Installed Tools or enter explicit executable paths.
3. Validate and save the profile.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
