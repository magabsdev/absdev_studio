# Query Console

## Purpose

Run SQL against the selected project database with visible output.

## How to use it

1. Verify the connection banner.
2. Start with read-only queries.
3. Use transactions for risky changes where supported.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
