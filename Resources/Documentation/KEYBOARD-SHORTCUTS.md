# Keyboard Shortcuts

- **Command-Shift-P** — Command Palette
- **Command-Shift-O** — Add Laravel Project
- **Command-1** — Overview
- **Command-2** — Development
- **Command-3** — Artisan
- **Command-4** — Terminal
- **Command-Shift-A** — AI Workspace
- **Command-Shift-M** — MCP Tools
- **Command-R** — Start or stop development
- **Command-Shift-R** — Refresh project capabilities
