# Mission Control

## Purpose

Mission Control summarizes project health, runtime, performance, Git, tests, security, and recent activity.

## How to use it

1. Select a project.
2. Open Overview or Product Studio.
3. Use cards to navigate to detailed tools.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
