# Upgrade Assistant

## Purpose

The upgrade assistant evaluates framework and package compatibility before a Laravel upgrade.

## How to use it

1. Create a branch.
2. Run compatibility analysis.
3. Apply changes incrementally and run tests.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
