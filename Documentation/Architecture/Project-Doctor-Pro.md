# Project Doctor Pro

## Purpose

Project Doctor evaluates runtime, Laravel configuration, packages, database, security, testing, and operational readiness.

## How to use it

1. Open Project Doctor.
2. Run diagnostics.
3. Fix errors before warnings and recommendations.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
