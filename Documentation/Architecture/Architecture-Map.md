# Architecture Map

## Purpose

Architecture tools visualise Laravel routes, controllers, models, services, jobs, events, and dependencies.

## How to use it

1. Refresh the Digital Twin.
2. Open Architecture Map.
3. Select a node to inspect the source relationship.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
