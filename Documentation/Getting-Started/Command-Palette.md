# Command Palette

## Purpose

Open the universal command palette to navigate features and run supported actions.

## How to use it

1. Open the palette from the toolbar or its keyboard shortcut.
2. Type a feature, command, or action.
3. Press Return to execute the selected result.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
