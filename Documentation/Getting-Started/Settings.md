# Settings

## Purpose

Configure application-wide preferences, optional AI services, appearance, and defaults for newly added projects.

## How to use it

1. Open Settings from the application menu.
2. Choose only the integrations you use.
3. Use project settings for runtime values that differ between projects.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
