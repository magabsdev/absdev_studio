# CI/CD

## Purpose

Deployment guidance covers repeatable test, build, migration, and release workflows.

## How to use it

1. Run tests and static checks.
2. Build production frontend assets.
3. Apply migrations with an explicit rollback plan.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
