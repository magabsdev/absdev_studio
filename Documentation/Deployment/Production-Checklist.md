# Production Checklist

## Purpose

Use a release checklist before deploying a Laravel application.

## How to use it

1. Back up data.
2. Verify environment and secrets.
3. Run tests, optimize caches, and confirm queue workers.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
