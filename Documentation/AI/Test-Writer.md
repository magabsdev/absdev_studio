# AI Test Writer

## Purpose

Generate Laravel unit, feature, API, Pest, PHPUnit, or Dusk test suggestions from project context.

## How to use it

1. Select a target feature or source file.
2. Choose the test style used by the project.
3. Run the generated tests and review failures.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
