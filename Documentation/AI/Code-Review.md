# AI Code Review

## Purpose

AI review uses selected project context to assess architecture, correctness, security, performance, and maintainability.

## How to use it

1. Select the correct project.
2. Choose a narrow review scope.
3. Review proposed changes before applying them.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
