# Project Memory

## Purpose

Project memory stores durable, project-specific context such as architecture decisions, terminology, and business rules.

## How to use it

1. Open Project Intelligence.
2. Review stored memory.
3. Remove obsolete entries so AI answers remain accurate.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
