# Scheduler

## Purpose

Inspect scheduled commands and tasks through Laravel scheduler tooling.

## How to use it

1. Open Scheduler.
2. Refresh the schedule list.
3. Verify the system cron or local scheduler process separately.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
