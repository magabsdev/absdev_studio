# Laravel Packages

## Purpose

Composer tools show installed packages and support package maintenance for the selected project.

## How to use it

1. Open Composer.
2. Refresh package information.
3. Review compatibility and tests before updates.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
