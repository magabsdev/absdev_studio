# Maintenance Mode

## Purpose

Control Laravel maintenance mode for the selected application.

## How to use it

1. Open Maintenance Mode.
2. Confirm the target environment.
3. Use a bypass secret where operationally appropriate.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
