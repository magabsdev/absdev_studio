# Cache Control

## Purpose

Cache Control provides explicit Laravel cache-clearing operations.

## How to use it

1. Open Cache Control.
2. Choose only the cache you intend to clear.
3. Review command output before continuing deployment work.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
