# Events and Listeners

## Purpose

Inspect Laravel events and listeners discovered in the project or exposed by Artisan.

## How to use it

1. Open Events & Listeners.
2. Refresh project capabilities after adding classes.
3. Use the architecture tools to trace downstream effects.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
