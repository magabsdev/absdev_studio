# Blade and Livewire

## Purpose

Frontend tools recognise Blade templates and Laravel frontend packages such as Livewire, Alpine, and Vite.

## How to use it

1. Open Frontend.
2. Review package detection.
3. Use project search for components and templates.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
