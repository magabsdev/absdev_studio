# Controllers

## Purpose

Inspect controllers through the architecture and route tools, then follow request handling into services and models.

## How to use it

1. Open Laravel Studio or Architecture Map.
2. Select Routes to identify controller actions.
3. Use project search to open the source file.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
