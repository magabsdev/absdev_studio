# Docker and Sail

## Purpose

ABSDEV Studio exposes container tools only when the relevant runtime is available and applicable to the selected project.

## How to use it

1. Start Docker or Apple Containers.
2. Select a project containing Sail when using Sail.
3. Open Containers or Sail from the navigator.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
