# ABSDEV Studio Help

This folder is the authoritative help system for ABSDEV Studio. The native Help Centre reads these Markdown files directly. Start with the Getting Started guide, then use the feature guides for day-to-day work.

## Main areas

- Project management and project-specific runtime configuration
- Laravel development tools
- Product Studio and Project Intelligence
- AI providers, MCP and the Knowledge Base
- Database, Git, deployment and diagnostics

## In the application

Open **Help Centre** from the tool navigator or the Help menu. Search matches titles, categories and full document contents.
