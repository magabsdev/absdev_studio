# Plugin SDK

## Purpose

The extension architecture is intended to support panels, commands, analyzers, runtime providers, and integrations.

## How to use it

1. Review the SDK documentation.
2. Declare only required capabilities.
3. Test extensions against a disposable project.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
