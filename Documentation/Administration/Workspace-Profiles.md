# Workspace Profiles

## Purpose

Workspace profiles preserve useful IDE layouts for development, database work, AI review, and debugging.

## How to use it

1. Arrange the workspace.
2. Save a named profile.
3. Restore the profile when changing tasks.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
