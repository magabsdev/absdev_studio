# Database Connections

## Purpose

Database tools use the selected project and its environment to resolve the intended connection.

## How to use it

1. Confirm the selected project.
2. Review .env connection values.
3. Avoid destructive operations until the displayed database is verified.

## Operational guidance

- The selected project is authoritative for project-aware commands.
- Review command output and diagnostics before proceeding.
- Use version control and backups before destructive operations.
