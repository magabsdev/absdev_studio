# ABSDEV Studio Brand Guide

## Product name

**ABSDEV Studio**

Use `ABSDEVStudio` only for technical identifiers such as Swift targets, executable names, schemes, folders, and bundle resources.

## Positioning

**The native development workspace for Laravel and modern local development.**

## Identity

- Primary mark: silver and blue `AS` monogram on graphite.
- Primary accent: studio blue.
- Supporting surfaces: graphite, slate, and macOS system materials.
- Product UI should use native macOS controls and system typography.

## Naming

- Product: ABSDEV Studio
- Executable/target: ABSDEVStudio
- Bundle identifier: `com.absdev.studio`
- Project document type: ABSDEV Studio Project
